# rd-132328-JS-Port

JS Port of minecraft version rd-132328 with three.js as renderer

![Preview](https://raw.githubusercontent.com/Fraeric123/rd-132328-JS-Port/main/screens/s1.png)

## Controls
* WASD   - Movement
* Space  - Jump
* R      - Random Teleport
* G      - Spawn Steve
* Enter  - Save .rdworld
* INSERT - Load .rdworld
* LMB    - Destroy block
* RMB    - Place block
